# Assistants
This guide explains how to customize the default Assistant instructions in the Jan application.

## Customize the Assistant's instructions
To customize the Assistant's instructions, follow these steps:
1. Select a **Thread**.
2. Click the **Assistant** tab.
3. In the **Instructions** field, input your desired instructions for the Assistant.
4. Toggle the **slider** to ensure these instructions are applied to all new threads. (Activate the **Experimental Mode** feature to enable this option.)
